## About The Project

<h4 align="left"> The Kapda Bazar e-commerce application is a fusion of innovative design and robust security. This report explores its architecture, emphasizing key features such as input validation, access control, encryption, two-factor authentication, session management, and audit logs. Through an in-depth exploration of each element, the report demonstrates how the application ensures data accuracy, user privacy, and system integrity. Four one-month sprints, following the agile Scrum methodology, facilitated iterative development and adaptation. Against this backdrop, Kapda Bazar emerges as a user-centric design with cutting-edge security practices, catering to modern e-commerce demands. </h4>

## Kapdabazar Features

* User Registration and Login
  - Create an account and log in securely to access personalized features.
* Diverse Fashion Product Range
  - Explore an extensive collection of fashion and clothing products.
* Shopping Cart
  - Easily add, remove, and manage selected items before proceeding to checkout.
* Review and Rating System
  - Provide feedback and ratings for purchased products to assist other users.
* Advanced Search and Filtering
  - Quickly find products using efficient search and filtering options.
* Personalized User Profiles
  - Customize your profile information and manage preferences.
* Responsive Design
  - Experience consistent usability across various devices, screen sizes, and operating systems.
* Input Validation
  - Implement stringent input validation techniques to ensure accurate data entry.
* Role-Based Access Control
  - Assign and manage different user roles with distinct permissions.
* Data Encryption
  - Safeguard sensitive user information through advanced encryption methods.
* Two-Factor Authentication
  - Enhance account security with an additional layer of verification.
* Session Management
  - Automatically log out users after a period of inactivity to prevent unauthorized access.
* Audit Logs
  - Keep a comprehensive record of system access and actions for accountability and security.
* Platform Compatibility
  - Compatible with different devices, screen sizes and OS.

## How To Use

```bash
# Clone this repository
$ git clone https://github.com/iambijayyy/KapdaBazar.git

# Go into the repository
$ cd KapdaBazar

# Install dependencies for frontend and start
$ cd frontend
$ npm install
$ npm start

# Install dependencies for backend and start
$ cd backend
$ npm install
$ npm start

# Access the application in your browser
http://localhost:3000/

```
